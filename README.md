# AR Target Shooter

A mobile AR game built with Unity 6 and AR Foundation where players shoot targets that spawn in their real-world environment. Built as part of my expansion from Lens Studio / Effect House AR development into Unity XR development.

## Demo
> Demo video coming soon

## Gameplay
- Point your phone at a flat surface (floor, table, desk)
- Targets automatically spawn in your real-world space
- Tap targets to destroy them and score points
- Beat the clock - how many can you hit in time?

## Tech Stack
- **Engine:** Unity 6
- **AR:** AR Foundation + ARCore (Android)
- **Language:** C#
- **Input:** Unity Input System
- **UI:** TextMeshPro

## Features
- [x] AR plane detection
- [x] Tap-to-destroy target mechanic
- [x] Real-time score counter
- [ ] Auto-spawn targets on detected planes
- [ ] 30 second countdown timer
- [ ] Game over screen with final score
- [ ] Restart flow
- [ ] Sound effects
- [ ] APK release

## Background
I'm an AR developer with experience in Lens Studio (JavaScript) and Effect House. This project marks my first step into Unity applying the same AR concepts I know (world tracking, plane detection, tap interaction) in a new engine and language.

## Run It Yourself
1. Clone the repo
2. Open in Unity 6 via Unity Hub
3. Ensure Android Build Support + AR Foundation packages are installed
4. Build and deploy to an ARCore-supported Android device
